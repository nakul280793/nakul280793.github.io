
# Fulai Mata Mandir — Soni Samaj — Static Website

## Structure
```
fulai-mata-mandir-site/
  index.html
  assets/
    css/style.css
    js/main.js
    images/logo.svg
    images/hero.svg
  pages/
    about.html
    darshan.html
    events.html
    visit.html
    donate.html
    contact.html
    volunteer.html
    faq.html
```

## How to Run
Open `index.html` in any browser. All assets are local.

## Customize
- Update address/phone in the footer.
- Replace `assets/images/*` with real photos.
- Edit aarti timings in `pages/darshan.html`.
- Hook payments and live darshan in `pages/donate.html` and topbar.

## Notes
- Mobile-friendly (menu drawer) and basic accessibility (semantic HTML, alt text, focus styles).
